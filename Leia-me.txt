PARA EXECUTAR O PROJETO VOCÊ VAI PRECISAR DO NODE JS

NODE JS https://nodejs.org/en/

DEPOIS DE INSTALADO VAI TER QUE INSTALAR O IONIC 5 COM NPM SEGUE OS COMANDO A BAIXO

npm install -g @ionic/cli

npm install

ASSIM PARA INSTALAR TODAS AS DEPENDÊNCIAS DOS PROJETO FEITO ISSO ENTRE COM O COMANDO 

ionic serve -l
